-------------------------------------
 como aprender inglês de forma fácil
-------------------------------------

Este livro contém uma série de dicas para que você
consiga aprender inglês de forma fácil. Este trabalho surgiu
à partir da experiência individual de um estudante que,
de forma auto-didata estoudou inglês por mais de dois anos,
durante este tempo ele descobriu novos métodos, ideias
inovadoras no aprendizado de idiomas.

O comparitlhamento desse material se tornou uma
necessidade e cremos, será de grande valia para
outras pessoas que estão estudando inglês.

  Para compilar este documento faça:

	make clean; make && make show

O make funciona devido ao arquivo makefile (veja o mesmo)

Outro detalhe, você deve ter instalado o pacote 'build-essential' assim como o LaTeX
